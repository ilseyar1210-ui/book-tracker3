# Book Tracker - Трекер прочитанных книг

**Автор:** Абзалова Эльвина Рамилевна
**GitHub репозиторий:** https://github.com/ilseyar1210-UI/book-tracker3

## Описание

Book Tracker — это графическое приложение на Python (Tkinter) для ведения списка прочитанных книг. Приложение позволяет добавлять книги с указанием названия, автора, жанра и количества страниц, фильтровать список по жанру и объёму, а также сохранять данные в JSON-файл для последующей загрузки.

## Установка и запуск

1. Убедитесь, что установлен Python 3.7 или выше
2. Скачайте файл `main.py` из репозитория
3. Запустите в терминале:
   ```bash
   python main.py
